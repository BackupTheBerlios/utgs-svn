Intro

 Back to ToC

Hi. My name is Radoslaw Kolasinski. In years 2001-2005 I've been involved as a Lead Programmer in several commercial game projects. The work presented here is derived from these projects. The components of this library were used commercialy in games of such types as: tycoon management, 3rd person advanture, or mini-games package for kids. On May 11th, 2005, I've received signed clerical permission to place these components into public domain. I believe this software can be used as good knowledge base.
